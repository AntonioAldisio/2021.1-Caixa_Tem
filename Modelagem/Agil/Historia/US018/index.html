    US018

    US018 - Eu como usuário do aplicativo desejo realizar pagamento via boleto.
    Regra/Critério de Aceitação Exemplo Questionamento
    Estar com a sessão iniciada no aplicativo.
    • O usuário efetuou login. - OK
    • O usuário não conseguiu efetuar login. - ERRO
    • O que torna a conta de um usuário inacessível?
    Estar com acesso liberado no aplicativo.
    • Solicitacao de acesso liberada. - OK
    • O aplicativo não conseguiu liberar acesso do usuário. - ERRO
    • Quais razões para não liberar o acesso do usuário?
    Ter o documento impresso o numéro de código em mãos.
    • O usuário tem o número do boleto. - OK
    • O usuário não sabe o número do boleto. - ERRO
